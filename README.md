# Guardian (FYP Project)

✅ **Guardian** is a React Native mobile app (built with Expo) for logging and visualising sensor activity and related incidents for my Final Year Project.

## What it is

Guardian displays sensor logs and activity in an accessible UI, with screens for viewing all activity and filtering logs by time ranges. It includes components for action buttons, stat cards, and header controls and uses local storage utilities for persisting logs.

## Project structure

- `App.js`, `index.js` — app entry points
- `src/app` — high-level app logic
- `src/components` — reusable UI components
- `src/screens` — screen components (e.g., `AllActivityScreen`, `AllLogsScreen`)
- `src/handlers` / `src/hooks` / `src/utils` — app logic, hooks, and utilities

## Setup & Run (Development)

1. Install dependencies:

   npm install

2. Start the app with Expo:

   npx expo start

   Then run on emulator or physical device via Expo Go - recommended.
